# NovaMarket Checkout Service

Documentación técnica inicial del servicio de checkout de NovaMarket.

## Estado del proyecto

La documentación se encuentra en construcción y debe mantenerse estable en la rama
main.

## Forma de trabajo

Los cambios deben ser revisables, trazables y fáciles de relacionar con una solicitud del
equipo.

## Colaboración externa



Las personas externas pueden proponer mejoras sin modificar directamente el repositorio

principal.

Toda contribución externa debe ser revisada antes de integrarse.
